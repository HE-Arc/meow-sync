# Meow-sync frontend
This is the frontend for Meow-sync, a playlist synchronization tool. It is built in Vue.js and uses the Composition API. The frontend communicates with the backend via REST API.

## Development
You will need node.js and uv installed to run the frontend in development mode. You can install them using your package manager or from their official websites.
To run the frontend in development mode, you'll first need to copy the .env.example file:
```bash
cp .env.example .env
```
Then you can install the dependencies and start the development server:
```bash
npm install
npm run dev
```
